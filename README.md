# Direct Webhook Test

This repository contains static informational pages required for Meta (Facebook / Instagram) app configuration.

The application is used exclusively for development and testing of webhook integrations
with Instagram and related Meta APIs.

No commercial services are provided.

---

## Purpose

This project exists to:
- Provide a public website URL for a Meta App
- Host Privacy Policy, Terms of Service, and User Data Deletion instructions
- Support development and testing of Instagram webhooks (including messaging)
